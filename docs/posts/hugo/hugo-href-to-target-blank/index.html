Hugoでリンクを新しいウィンドウで開く

hugo 
 2018-10-16

リンクは新しいウィンドウで開いて欲しい派なんだけど、Hugoではデフォルトで通常のリンクの開きかた(?)をする。

hrefではtarget='_blank'を指定すれば良いのだが、markdownでどうすれば良いのか調べた。 Hugoでは次のようにconfig.tomlするといける。

[blackfriday]
   hrefTargetBlank = true

簡単！
